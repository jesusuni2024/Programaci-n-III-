# Lista Doblemente Enlazada en Java

Este proyecto implementa una estructura de datos de **lista doblemente enlazada** utilizando Java.

## Funcionalidades

- Insertar al inicio
- Insertar al final
- Recorrer hacia adelante
- Recorrer hacia atrás
- Mostrar tamaño de la lista
- Verificar si la lista está vacía
- Buscar elemento por valor
- Buscar elemento por índice
- Borrar un elemento
